# SQL Practice - SQLZoo Exercises

This repository contains my solutions to SQLZoo exercises, organized by topic. The goal of this project is demonstrate my practical SQL skills: filtering, aggregations, joins

## Data source
All queries are based on the SQLZoo datasets:
- World database (countries, population, area, GDP)
- Nobel Prize database

## Topics covered
- Basic SELECT statements
- Filtering with WHERE, IN, BETWEEN
- Sorting and limiting results
- Aggregations (COUNT, SUM, AVG)
- GROUP BY and HAVING
- JOINs between tables
- Subqueries
